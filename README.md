# GEM LENS

**La primera herramienta argentina de grading estimado por IA para Pokémon TCG, Yu-Gi-Oh!, Magic y más.**

Valuá tu carta en 30 segundos: autenticidad, GLS Score (0-10), GLP (0-500), precio en USD y ARS al blue, ROI de grading y consejo de inversión local.
